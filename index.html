Welcome to BeatBox

BeatBox is a free, open source four-channel DJ application.
Suited for both beginners as well as advanced DJs,
BeatBox pairs an easy-to-use / minimalistic UI
with industry-standard features.

BeatBox utilizes third-party components such as
Wavesurfer.js and Font-Awesome stylesheets.
More information on these tools
at the following links:

https://wavesurfer-js.org/
https://fontawesome.com/



Click anywhere to start mixing.
